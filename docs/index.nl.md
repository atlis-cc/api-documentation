# Huis

## Informatie
De Atlis API is een openbare API die toegankelijk is voor iedereen met een account en geregistreerde API-sleutel. *Momenteel is het alleen beperkt tot gebruikersaccounts.*

* Eindpunten

* Uitzonderingen
